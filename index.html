Cómo agregar documentos

Este sitio no tiene panel de administración: los documentos se publican directamente en el repositorio.

  1. Coloca el archivo PDF dentro de la carpeta pdfs/.
  2. Abre documentos.json y agrega un nuevo bloque con título, descripción, fecha y el nombre del archivo.
  3. Guarda y sube los cambios (commit + push si usas GitHub Pages). El documento aparecerá automáticamente en la lista.

Revisa el archivo README.md del proyecto para ver un ejemplo completo.
